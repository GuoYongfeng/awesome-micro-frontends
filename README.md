# Awesome Micro Frontends

![awesome](./media/awesome.svg)  ![zh-cn](./media/zh-cn.svg)  ![license](./media/creative-commons.svg)

微前端是一种类似于微服务的架构，它将微服务的理念应用于浏览器端，即将 Web 应用由单一的单体应用转变为**多个小型前端应用聚合为一的应用**。各个前端应用还可以**独立运行**、**独立开发**、**独立部署**。
